CMS has opened up a large amount of data for public access and analysis. Below is a portal to the 2014 Medicare Utilization dataset that includes a 5% sample of all Medicare provider claims on record. For this project, we look claims that relate to biopsy samples. In this case, we are interested in setting up a central pathology lab that each state can ship their samples to for analysis. The key is selecting the state that will minimize the total distance each sample must travel.

Each instance in which a patient is biopsied constitutes a case in which a sample needs to be shipped from a clinic to a reference lab for analysis. The following Tableau table provide a vizualization of the frequency of each of these instances in each state:
